# RAG 智能问答系统

基于检索增强生成(RAG)的智能文档问答系统，专门用于运维工程师知识检索和智能问答。

## 项目简介

本系统集成了文档处理、语义搜索和大语言模型，可以基于知识库文档提供准确的问答服务。系统支持Word文档作为知识来源，并提供智能化的问答能力。

## 项目结构

```
rag_system/
├── src/                   # 源代码目录
│   ├── processors/        # 文档处理模块
│   ├── embeddings/        # 向量嵌入模块
│   ├── llm/              # LLM客户端模块
│   └── rag/              # RAG系统核心模块
├── config/                # 配置文件目录
├── data/                  # 数据目录
│   └── knowledge_base/    # 知识库文件
└── main.py               # 主入口文件
```
